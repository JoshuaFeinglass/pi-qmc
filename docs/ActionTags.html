EFieldAction

   < EFieldAction strength="50000 V/cm" dir="z"  width="10 nm" />

dir = "x" "y" or "z" , width is region centred at 0 where finite field is generated.

  • model

  • "SHONodes"
  • WireNodes"
  • "GSSNode" or "GroundStateSNode"
  • "species"
  • double temperature
  • bool noNodalAction

    SHOAction

    Action for a Simple Harmonic Oscillator (SHO) potential, using the exact density matrix (For a derivation, see Feynman, Statistical Mecahnics (1972).)

     < SHOAction omega="30 meV" ndim="3" species="hole"/>

    Attributes: omega - sets hbar*omega of the SHO Action.   ndim - defines the number of dimensions for the action, if left out the action will default to no. dimensions in the simulation.   species - Sets action to work on a species.
