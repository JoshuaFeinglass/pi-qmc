Detailed Description

Unnormalized density matrix for 2s and 2p orbital.

To avoid negative regions, this does not include the radial node in the 2s orbital, and there is enough 2s contribution to avoid an angular node (provided ).

where . and controls the strength angular dependence ( for s-only).
