Detailed Description

Class for fixed-phase action.

In the fixed-phase approximation, the phase of a trial function (defined by the PhaseModel) contributes to the hamiltonian,

This hamiltonian acts on the same slice as , but references coordinate positions on slice .

This action needs careful treatment, since the phase of the trial function is discontinuous for real-valued functions. Rather that relying solely on the value of the phase gradients a the ends of the time interval, we fit a cubic function to the phase gradients and values, and integrate along a straight trajectory. Let be the partial derivative of along the direction from to . We fit the phase and gradient of the phase to the value and gradient of the phase at points and , using a cubic polynomial for the gradient along the link and linear interpolation for gradient components perpendicular to the link directions.
