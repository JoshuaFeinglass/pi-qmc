Detailed Description

Select a trial move for beads by free particle sampling.

This class treats periodic boundary conditions more carefully, which is necessary for very small box sizes.

The difficulty, is that the bisection algorithm has 2**NDIM midpoints when you consider periodic boundary conditions. If these are not sampled carefully, some windings or permutations may be missed, raising the energy.
