Detailed Description

Free particle nodes for new fermion algorithm.

It does this by returning HUGE_NUMBER for the action value from getActionDifference and getTotalAction methods.

The free particle slater determinant is a slater determinant of PeriodicGaussians with width , or .

There is a nodal action for paths that come close to the node but do not cross it. As in Ceperley's article, we approximate this fermion action as

For the virial estimator, we need the nodal force, defined as the negative gradient of the nodal action for coordinates at slice i. From the chain rule we get

where the indicies refer to particles.

Before sampling, once we know the section, a call to initialize evalutates and stores inverse slater matricies and the distances to the nodes for each slice in the section, including the end slices. The evaluateUpdate method calculates the new slater matrices and (on the last level) the new distances to the nodes. Evaluate update returns the difference in nodal action (HUGE_NUMBER if a node was crossed). If the move is accepted, acceptLastMove updates the stored matrices and nodal distances.
