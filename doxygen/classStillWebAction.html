Detailed Description

Class for calculating the primitive action for the Stillinger-Weber potential.

Taken from Stillinger and Weber, PRB 31, 5262-5271 (1985), online at http://link.aps.org/abstract/PRB/v31/p5262. The pairwise interaction is a function of separation,

and zero for larger separations.

There is an angular interaction between bonds,

Si-Si lattice paramters is 10.246 Bohr radii (T=0K).

Bug:
Assumes only one particle is moving.
