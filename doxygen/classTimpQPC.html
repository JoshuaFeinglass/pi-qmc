Detailed Description

Class for calculating the action for a model of a quantum point contact.

This is a parameterized model of the repulsive potential for gates with potential , width w, a gap length l, a distance z above the 2DEG. From Temp (eq. 1) in Semiconductors and Semimetals 35, 113-190 (1992).

where

We set the default values to w=0.01 um, l=0.03 um, vG=-1V, and z=0.01 um.
